# Youtube-Search-Reactjs


## Installation

Install all dependencies:

npm install

Run app

npm start

**Create an API KEY**, then create a `key.js` file inside the `src` folder, and write this:

javascript
const API_KEY = 'YOUR_PERSONAL_KEY';
module.exports = API_KEY;
